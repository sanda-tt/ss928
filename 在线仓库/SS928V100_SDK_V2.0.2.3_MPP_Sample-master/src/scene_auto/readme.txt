
This readme introduce how to start scene_auto of the sensor, but does not guarantee the parameters for the scenario:
take hy_s0603 as an example:

1.1 hy_s0603 linear mode
./sample_scene param/sensor_hy_s0603
init success
1.scene start
2.scene pause and reset media route
3.scene resume and set a new media type
4.exit the sample
1
When We Start SceneAuto, we neen to set video Pipe mode
Please input videomode index, without which we couldn't work  effectively.
0
videomode type has already been input.
The sceneauto is started already.
1.scene start
2.scene pause and reset media route
3.scene resume and set a new media type
4.exit the sample
