TinySam模型可以通过以下代码完成快速推理
```
#include "sam_predictor.h"

using namespace Infer;

int main()
{
    EnvInit();
    SAMBoxPredictor predictor;
    predictor.loadModel(modelPaths);
    predictor.setImage(image); // 编码图像数据
    predictor.predict(boxes);  // 设置检测框进行分割模型掩码解码
    EnvDeinit();
    return 0;
}
```
备注：上述C++代码依赖的动态库与文件位于[/samples/common](https://gitee.com/HiSpark/modelzoo/tree/master/samples/common)和[/TinySam/src/](https://gitee.com/HiSpark/modelzoo/tree/master/samples/built-in/segmentation/TinySam/src)目录下，编译相关配置参考[CMakeLists.txt](https://gitee.com/HiSpark/modelzoo/blob/master/samples/built-in/segmentation/TinySam/src/CMakeLists.txt)。
