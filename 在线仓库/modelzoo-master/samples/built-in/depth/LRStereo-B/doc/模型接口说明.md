# 基于LiteAndRobustStereo网络API
LiteAndRobustStereo为一个双目深度估计模型。它在开源模型(Raft-Stereo)的基础上做了大量的模型结构改进和重训。具体功能为输入标定好的左右目图像以及相关的相机参数，获得左目图像对应的深度图。

# om说明
模型1：LiteAndRobustStereoS_384x1248_release.om
说明：输入格式为384x1248分辨率的左右目图片，推理后输出为1x384x1248格式的视差值。

模型2：LiteAndRobustStereoS_480_640_release.om
说明：输入格式为480x640分辨率的左右目图片，推理后输出为1x480x640格式的视差值。

模型3：disp2depth_384x1248.om
说明：输入为模型1推理后输出，1x384x1248格式的视差值，输出为1x384x1248格式的距离值。

模型4：disp2depth_480x640.om
说明：输入为模型2推理后输出，1x480x640格式的视差值，输出为1x480x640格式的距离值。

# 接口说明
1、int32_t EnvInit(const std::string& configPath)；
		设备初始化。

​		输入：acl.json文件路径。

​		输出：0表示加载成功，1表示加载失败。

2、int32_t EnvDeinit()
		设备去初始化。

​		输出：0表示加载成功，1表示加载失败

3、 int32_t Model::Load(const std::string& modelPath, ModelType modelType)

​		加载om模型。

​		输入：modelPath，om模型路径。

​					modelType，模型类型，比如模型1、2为XStereo；模型3、4为XStereoDis

​        输出：0表示加载成功，1表示加载失败。

4、std::vector< std::vector<Tensor> > Infer(const std::string& filePath, FileType fileType = JsonFile);

​		模型推理

​		输入：filePath待推理图片或者数据集列表；fileType 文件类型，json文件或者单张图片路径

​		输出：推理结果

5、 std::vector<Tensor> Infer(std::vector<Tensor>& tensors);

​		模型推理

​		输入：tensors推理模型输入。

​		输出：推理结果

6、std::vector<Tensor> Infer(std::vector<Tensor>& tensors, std::string filePath);

​		模型推理

​		输入：tensors推理模型输入。filePath输入文件路径

​		输出：推理结果

7、int32_t Model::Unload() { }

​		模型卸载

​		输出：0表示加载成功，1表示加载失败。
