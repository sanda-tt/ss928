本目录为`SDK`软件相关文件，包括`lib`库，驱动源码，`sample`源码等。

- 支持`clang`和`gcc`两种工具链，分别对应`ss928v100_clang`和`ss928v100_gcc`

  |                   | 编译器       | 运行时库          |
  | ----------------- | ------------ | ----------------- |
  | `ss928v100_clang` | `llvm 15.04` | `musl libc 1.2.5` |
  | `ss928v100_gcc`   | `gcc 12.3.0` | `glibc 2.38`      |
